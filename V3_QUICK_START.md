# V3 可跟品产品 - 快速开始

## 5 分钟快速上手

### 1. 启动服务
```bash
cd /Users/doi/Desktop/amazon
python3 shopify_api_server.py
```

### 2. 打开页面
浏览器访问: http://127.0.0.1:8001/v3

### 3. 加载数据
点击页面顶部的 **"快速加载"** 按钮

### 4. 查看可跟品产品
点击 **"🎯 可跟品产品"** 标签页

### 5. 生成 AI 调研
点击任意产品行的 **"AI 调研"** 按钮，提示词自动复制到剪贴板

### 6. 分析产品
将提示词粘贴到 Claude 对话框，获得 8 维度深度分析

## API 快速测试

### 获取可跟品产品（评分 ≥ 50）
```bash
curl "http://127.0.0.1:8001/api/v3/followable-products?min_score=50&limit=5"
```

### 生成 AI 调研提示词
```bash
curl -X POST "http://127.0.0.1:8001/api/v3/generate-research-prompt?product_url=https://example.com/products/handle"
```

## 提示词使用示例

1. 点击"AI 调研"按钮（自动复制）
2. 打开 Claude 对话框
3. 粘贴提示词（Ctrl+V / Cmd+V）
4. 发送给 Claude
5. 获得完整的 8 维度分析报告

## 常见问题

**Q: 没有数据怎么办？**  
A: 先点击"快速加载"或"刷新扫描"生成数据

**Q: 评分都很低？**  
A: 降低"最低评分"到 40-50，然后点击"刷新"

**Q: AI 调研按钮点击无反应？**  
A: 检查浏览器是否支持 Clipboard API（需要 HTTPS 或 localhost）

**Q: 想要更多产品？**  
A: 修改 URL 参数 `limit=100` 增加返回数量
